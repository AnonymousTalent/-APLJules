# LightningEmpire

This is the main monorepo for the LightningEmpire project, containing all modules for the system.

## Modules

-   `/server`: Backend API server (Order processing, payments)
-   `/ai-core`: Core AI logic modules (Little Flash 4.1 / 5)
-   `/tasks`: Automated tasks and workers (Routing, Telegram Bot)
-   `/db`: Database schema and migration tools
-   `/cli`: Command-line interface for managing the empire
-   `/mobile-app`: (Future) Expo-based mobile application
